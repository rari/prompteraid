About PrompterAid

PrompterAid is a visual reference tool for users of the NijiJourney and Midjourney AI models. It displays pre-generated style references (--sref codes) using a consistent series of mermaid-themed images, all created with the same seed and a simple mermaid prompt. This uniform approach makes it easy to compare styles directly and combine them to fine-tune your prompts with precision.

How to use PrompterAid

  • Hover over the model name (e.g., "Niji 6") in the subheader to switch between NijiJourney and Midjourney models. The appropriate version tag (--niji 6 or --v 7) will be automatically added to your prompt.
  • Enter your base prompt in the text field.
  • Click one or more images to select their styles.
  • The final prompt, including --sref and the correct version tag, is auto-generated.
  • Click or press 'C' to copy the prompt to your clipboard.

Platform

  • Discord: Paste the prompt into the MidJourney or NijiJourney Discord bot, just like any other prompt.
  • Website: Use the generated prompt directly on the Midjourney or NijiJourney website.

Both bots and websites can accept the same --sref, --niji 6 and --v 7 parameters!

How to Use Style References

An --sref code is a unique numeric ID that points to a predefined visual style stored by NijiJourney controlling the look, mood, and aesthetic of an image (like color, lighting, or composition) without needing an image upload.

The code represents a specific direction within MidJourney's multidimensional style space. Think of your prompt as starting at the center of a sphere. Each style reference is a vector‚Äîan arrow‚Äîpointing outward toward a particular visual aesthetic.

The --sw parameter controls how far along that vector your prompt travels, adjusting the strength of the style influence.

Advanced Options:
  • Multiple styles: Apply more than one style by listing multiple --sref codes to blend their visual influences.
  • Weighted styles: Add ::weight to each --sref code to control how strongly each one affects the output.
  • Style strength (0‚Äì1000): Use --sw to adjust the overall intensity of the style reference's influence on the image.

Learn more about this topic in the official documentation.

Menu Tools

  • Deselect: Clear all selected images.
  • Refresh: Shuffle the order of all images and randomize their starting view.
  • Favorites View: Show only your starred images.
  • Selected View: Show only your currently selected images.
  • Search: Search for images by their number(s). Use spaces to search for multiple numbers (e.g., "1 2 7").
  • Randomize: Randomly select an unselected image.
  • Copy: Copy the final generated prompt to your clipboard.

Image Tools

  • Favorite: Add the image to your favorites collection.
  • Rotate: Cycle through other visual examples from the same style reference code.
  • Select: Left-click the image to add its style code to your prompt. Selected images are shown with a colored border.
  • Link: Share a direct link to any style reference!
  • Weight: Adjust the relative importance of each code.

Hints

  • We save your preferences and favorites to your local browser storage!
  • You can add extra parameters to your base prompt like --ar or --p.
  • Use the rotate button to see different examples of the same style.
  • Try combining complementary styles for unique results.
  • Toggle dark mode for better nighttime viewing.
  • You can close this info menu by clicking on the colorful bar at the top of it.
  • Keep prompts focused on subject, not style. Let --sref handle the visual tone.
